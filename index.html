Light and Sound Memory Game!!

Welcome to the Light and Sound Memory Game.

When the game starts, a random button will light up. Your goal is to remember which button that was, and click it. There are eight rounds, with the first round having one button light up, and an extra button being added to the randomly generated sequence each round.

If you get can through all eight rounds successfully, you win. If not, you lose. Best of luck :)

Difficulty:

Select your difficulty! By default, you are set to easy. Each increase adds an extra four rounds.
